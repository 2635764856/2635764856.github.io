关于我

关于我#

欢迎来到我的个人空间。我是 曾俊涛，一名充满热情的后端开发工程师，致力于构建高效、可靠和可扩展的系统。

个人简介#

我是一名来自四川成都的开发者，现就读于西华大学物联网工程专业（本科一批），专业排名前 10%。我对后端开发、微服务架构和系统设计充满热情，相信通过代码可以解决实际问题，改善人们的生活。

我的开发哲学是：简洁、高效、可维护。无论是设计数据库架构还是实现业务逻辑，我都力求在功能性和代码质量之间找到最佳平衡。

核心技能#

编程语言与框架#

我掌握多种编程语言，包括 C/C++、Python 和 JavaScript，并具备 Java 桌面端与 Kotlin 移动端的开发经验。在后端开发方面，我对 Node.js + Express 框架有深入的理解，能够设计和实现符合 RESTful 标准的 API。

数据库与缓存#

我熟悉 MySQL 数据库的表结构设计与优化，能够处理百万级数据的建模。同时，我具备使用 Redis 进行缓存优化的实践经验，能够显著提升系统性能。

微服务与消息队列#

我了解微服务架构的设计原则，具备 RabbitMQ 异步消息处理的实践经验。我能够设计高效的消息队列系统，处理复杂的业务流程。

安全与认证#

我具备 Spring Security 和 JWT 等用户认证与授权机制的应用经验，能够实现基于角色的访问控制（RBAC）。

实时通信#

我具备 WebSocket 实时通信的实践经验，能够构建实时数据推送系统。

运维与部署#

我熟悉 Linux 环境操作，具备 Docker 容器化部署与 CI/CD 流程的经验。我能够从本地开发环境到生产环境的完整部署流程。

测试与调试#

我熟悉 Postman 和 JMeter 等测试工具，能够进行接口调试与性能测试。

项目经历#

农产品智能仓储系统#

时间: 2024.07 – 2025.01 | 角色: 后端开发 | 成就: 软件著作权（登记号：2025SR0210727）

这是一个大规模的微服务项目，我参与了从架构设计到部署的全过程。我负责开发用户、库存、环境监控和预警模块。项目的核心挑战包括：

  • 百万级数据库建模：设计了支持批次管理、保质期和库存状态的复杂数据结构，实现了 FIFO 出库策略。
  • 权限控制：使用 Spring Security 和 JWT 实现了基于角色的访问控制（RBAC）。
  • 实时监控：设计了环境监控 API，能够接收嵌入式设备上传的数据，并通过 WebSocket 实时推送给前端。
  • 性能优化：集成了 Redis 缓存和 RabbitMQ 异步处理，显著提升了系统性能。
  • 容器化部署：编写了 Dockerfile 和 Docker Compose 配置，支持 CI/CD 流程。

基于 FS-MP1A 的元宇宙智能农业系统#

时间: 2024.06 - 2024.07 | 角色: 后端开发

在这个项目中，我负责智能农业系统的设备配置与软件之间的连接。我全程参与了项目的设计、开发和测试，协调不同模块之间的共同工作。

基于 Android 的智能课堂考勤系统#

时间: 2025.01 - 2025.03 | 角色: 后端开发 | 定位: 毕业项目

这是我的毕业项目，一个涵盖学生、教师和管理员三端的综合系统，具备位置定位和人脸识别功能。我的主要职责包括：

  • 后端架构：使用 Node.js 和 Express 搭建了整个后端系统，设计了符合 RESTful 标准的 API。
  • 数据库优化：进行了 MySQL 数据库建模，并通过连接池优化支持高并发。
  • 实际测试：系统经过校园 11 个班级（300 人）的实际教学环境测试，表现出良好的并发性能和抗压能力。
