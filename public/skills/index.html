技能与工具

技能与工具#

这个页面详细列出了我的技能树和日常使用的开发工具。我相信持续学习和不断提升自己的能力是成为一名优秀工程师的关键。

编程语言#

精通#

C/C++ 是我最早学习的编程语言，也是我最熟悉的语言之一。我使用 C/C++ 进行系统级编程、算法实现和性能关键的应用开发。

Python 是我用于快速原型开发、数据处理和脚本编写的首选语言。它的简洁语法和丰富的库生态让我能够快速实现想法。

JavaScript 是我在 Node.js 后端开发中的主要语言。我熟悉异步编程、事件驱动架构和函数式编程范式。

熟悉#

Java 是我用于企业级应用开发的语言。我具备使用 Spring 框架进行后端开发的经验。

Kotlin 是我用于 Android 移动端开发的语言。作为 Java 的现代替代品，Kotlin 提供了更简洁的语法和更强大的功能。

后端框架与技术#

Node.js + Express#

我对 Node.js 和 Express 框架有深入的理解。我能够：

  • 设计和实现 RESTful API
  • 处理异步操作和回调函数
  • 实现中间件和错误处理
  • 进行性能优化和调试

Spring Boot#

我具备使用 Spring Boot 进行微服务开发的经验。我能够：

  • 配置和使用 Spring Security 进行身份验证和授权
  • 实现 RESTful 服务
  • 集成数据库和缓存
  • 进行单元测试和集成测试

数据库技术#

MySQL 是我最常用的关系型数据库。我能够：

  • 设计规范化的数据库架构
  • 优化查询性能
  • 使用索引和查询优化技术
  • 管理事务和并发控制

Redis 是我用于缓存和会话管理的首选。我能够：

  • 使用 Redis 数据结构（字符串、列表、集合、有序集合等）
  • 实现缓存策略和过期管理
  • 使用 Redis 进行分布式锁
  • 监控和优化 Redis 性能

消息队列#

RabbitMQ 是我用于异步消息处理的主要工具。我能够：

  • 设计消息队列架构
  • 实现生产者和消费者
  • 处理消息重试和死信队列
  • 监控消息队列性能

实时通信#

WebSocket 是我用于实现实时数据推送的技术。我能够：

  • 建立 WebSocket 连接
  • 实现双向通信
  • 处理连接管理和心跳检测
  • 优化实时数据传输

安全与认证#

Spring Security#

我具备使用 Spring Security 进行身份验证和授权的经验。我能够：

  • 实现基于角色的访问控制（RBAC）
  • 配置安全过滤器链
  • 处理登录和注销流程
  • 实现权限检查

JWT (JSON Web Token)#

我能够使用 JWT 实现无状态的身份验证。我能够：

  • 生成和验证 JWT
  • 处理 token 刷新
  • 实现 token 过期管理
  • 确保 token 的安全性
